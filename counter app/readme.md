# Counter App

A simple web-based counter application built with HTML, CSS, and JavaScript. This app allows users to increment a counter and save previous entries.

## Features

- **Increment Counter**: Click the "Increment" button to increase the count by 1.
- **Save Entries**: Click the "Save" button to save the current count and reset the counter to 0.
- **Display Previous Entries**: View all saved entries in a list.
- **Welcome Message**: Personalized welcome message displayed at the top.

## How to Use

1. Open `index.html` in your web browser.
2. Click the "Increment" button to increase the counter.
3. Click the "Save" button to save the current count and start a new count.
4. Previous entries will be displayed below the save button.
